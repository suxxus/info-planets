# Info-planets
***Emberjs-exercise***

* Emberjs
* D3js

## Clone the repo & install
```
git clone https://github.com/suxxus/info-planets.git
cd ./info-planets
npm install
bower install
```
## Usage
* npm run update
* delete .git
* git init
* node server/server.js
* cd ./client  
* ember s --proxy http://localhost:8000 (start an ember cli)
